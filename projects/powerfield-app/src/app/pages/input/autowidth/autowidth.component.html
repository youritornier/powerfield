API reference for Powerfield Autowidth directive

powAutowidth

The Autowidth Powerfield directive automatically adjusts a text input's width to its content.

Properties

Name Description
@Input() powExtraWidth: number Extra space added to the input's width (in pixel).
@Input() powMaxWidth: number Maximum size of the input (in pixel). If maximum size is 0, it is ignored.
@Input() powMinWidth: number Minimum size of the input (in pixel). If minimum size is 0, it is ignored.
