Getting started

Welcome to Powerfield! Powerfield helps you with common needed Angular input directives.

This guide shows you how to start using Powerfield in your Angular projects.

Step 1: Install the Powerfield package

To install the Powerfield package in your project using npm, open a terminal/console window and enter the following command: 

  npm install powerfield

Step 2: Import Powerfield to your module

To import the installed package to your module, open your module file and edit it as follow: 

  import { PowerfieldModule } from 'powerfield';

@NgModule({
  ...
  imports: [
    ...
    PowerfieldModule,
  ],
  ...
})
...

Next steps

Now that you've installed and imported Powerfield, you can use every provided directives directly in your component templates.

For example, you can use the powAutoWidth directive as follow: 

  <input type="text" powAutoWidth [powExtraWidth]="20" />
